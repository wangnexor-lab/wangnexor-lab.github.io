Linux 操作系统介绍

Linux是一款开源的操作系统，它的内核由Linus Torvalds于1991年创建。Linux系统的设计目标是提供一个稳定、安全、高效的操作系统，同时也提供了一个开放的平台，使开发人员可以自由地开发和修改操作系统的代码。Linux系统常常被用于各种服务器端应用，如Web服务器、数据库服务器等。在我们日后pwn的过程中，会用到很多Linux系统的指令操作，如文件操作、进程管理、网络配置等。因此，掌握Linux系统的指令操作是非常必要的。（ps：MacOS系统与Linux指令高度相似，因为MacOS是Unix内核,而Linux是类Unix系统）

Linux 操作系统的特点

  • 一切皆文件：在Linux系统中，所有的设备、文件、目录、进程等都被视为文件。这使得Linux系统的文件操作非常灵活和方便。
  • 权限严格：在Linux系统中，每个文件和目录都有自己的权限设置。这些权限可以控制谁可以读取、写入或执行文件。
  • 命令行就是本体：图形界面都是装饰，命令行才是Linux系统的核心。

Linux 常用版本（学习pwn必备）

  • Ubuntu：新手友好，但不适用于网安学习。
  • Kali Linux：专为渗透测试和网络安全而设计，包含了许多常用的渗透测试工具。
  • CentOS/Rocky：企业级Linux发行版，稳定性和安全性较高。
  • Parrot Security OS：基于Debian的操作系统，专注于网络安全和渗透测试。
  • Debian：很多服务器用，应该有所了解。
  • Arch：Linux终极形态，一切都可以自定义，适合有一定Linux基础的人，让系统变成你喜欢的样子，鼠标将变成摆设，键盘将变成你的一切。

Linux 文件系统

  • 目录结构：Linux系统的目录结构是一个树状结构，根目录(root)为“/”。其他目录都是根目录的子目录,如“/bin”、“/etc”、“/home”等。
  • 文件权限：每个文件和目录都有自己的权限设置。这些权限可以控制谁可以读取、写入或执行文件。
  • 文件操作：在Linux系统中，文件操作非常灵活和方便。可以使用命令行或图形界面来进行文件操作。

绝对路径和相对路径

在Linux系统中，每个文件和目录都有自己的路径。路径可以分为绝对路径和相对路径。

  • 绝对路径：从根目录开始的完整路径，如“/home/user”。
  • 相对路径：相对于当前工作目录的路径，如“../user”。这里的“../”表示上一级目录，而“./”则可以表示当前目录，例如：“./user”表示当前目录下的user目录。
  • 特殊路径：
    • “~”：表示当前用户的主目录，如“/home/user”。
